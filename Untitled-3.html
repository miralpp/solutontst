Theoretical Foundation

Projectile motion is described using basic equations derived from Newton's laws:

  • Horizontal Motion: \( x(t) = v_0 \cos(\theta) t \)
  • Vertical Motion: \( y(t) = v_0 \sin(\theta) t - \frac{1}{2} g t^2 \)

These equations form the basis for analyzing motion under varying initial conditions.

Range Analysis

The range of the projectile is maximized under ideal conditions at \( \theta = 45^\circ \):

\( R = \frac{v_0^2 \sin(2\theta)}{g} \)

Practical Applications

This model can be adapted to various real-world situations:

  • Sports: Calculating the arc of a ball.
  • Engineering: Optimizing rocket trajectories.
  • Uneven Terrain: Incorporating initial height \( h \):
\( R = \frac{v_0 \cos(\theta)}{g} \cdot \left( v_0 \sin(\theta) + \sqrt{(v_0 \sin(\theta))^2 + 2gh} \right) \)

Implementation

You can simulate projectile motion using Python:

import numpy as np
import matplotlib.pyplot as plt

def projectile_motion(v0, theta, g=9.81, h=0):
    theta_rad = np.radians(theta)
    t_flight = (v0 * np.sin(theta_rad) + np.sqrt((v0 * np.sin(theta_rad))**2 + 2 * g * h)) / g
    t = np.linspace(0, t_flight, num=500)
    x = v0 * np.cos(theta_rad) * t
    y = v0 * np.sin(theta_rad) * t - 0.5 * g * t**2
    return x, y

v0 = 20
theta = 45
x, y = projectile_motion(v0, theta)

plt.plot(x, y)
plt.title("Projectile Motion")
plt.xlabel("Horizontal Distance (m)")
plt.ylabel("Vertical Distance (m)")
plt.grid()
plt.show()
